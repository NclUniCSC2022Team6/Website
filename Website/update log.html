Update Log

Version 1.0.0: Released the app in its first iteration. Features include:

  • Searching for rooms within the USB, including searching for tutors by name and room number.
  • Navigation drawer that allows you to select a floor and look at the rooms on any given floor the user selects.
  • Section detailing information about the building
  • An open day page which gives the user a small tour of the building
